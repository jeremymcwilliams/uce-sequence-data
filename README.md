# UCE Sequence Data Pipeline

Steps for setting up a bioinformatics pipeline for ultraconserved element (UCE) data, adapting the work of Fabián Alfonso García to our HPC (SLURM) environment.

## Project Overview

This pipeline processes Illumina paired-end sequencing data from a spider phylogenomics study. Raw reads were generated by RAPiD Genomics (project LAC_203801) across 48 specimens representing multiple spider taxa. The goal is to identify and extract UCE loci for phylogenetic analysis.

## Pipeline Steps

| Step | Script | Tool | Description |
|------|--------|------|-------------|
| 1 | `01_fastqc_raw.sh` | FastQC | Quality check on raw combined reads |
| 2 | `02_trimmomatic.sh` | Trimmomatic | Adapter trimming and quality filtering |
| 3 | `03_fastqc_trimmed.sh` | FastQC | Quality check after trimming |
| 4 | `04_spades.sh` | SPAdes | De novo genome assembly |
| 5 | `05_organize_contigs.sh` | bash | Organize contigs for Phyluce |
| 6 | `06_phyluce_mapping.sh` | Phyluce | Match contigs to UCE probes |
| 7 | `07_phyluce_get_counts.sh` | Phyluce | Count UCE loci per sample |
| 8 | `08_phyluce_get_fastas.sh` | Phyluce | Extract UCE sequences |
| 9 | `09_fuse.sh` | FUSe | Align, trim, and filter alignments |

## Requirements

- SLURM-based HPC (Ubuntu)
- Miniconda3 with two environments:
  - `Bioinfo` — FastQC, Trimmomatic
  - `phyluce-1.7.3` — Phyluce, FUSe
- SPAdes 4.2.0 (installed in PATH)
- Spider UCE probe file: `probes/RTA-v3-probe-combine-spider-color-DUPE-SCREENED.fasta` (included in this repo)
